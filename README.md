## 📋 How to Use openclaw-youtube-archiver

### Step 1: Prepare your YouTube playlist URL

- Go to YouTube and open the playlist you want to archive.
- Copy the URL from your browser's address bar.

### Step 2: Paste the playlist URL into the app

- In the openclaw-youtube-archiver window, find the box labeled "Playlist URL."
- Paste the copied URL there.

### Step 3: Choose output folder

- Pick where you want to save the notes and videos.
- Click the "Browse" button to select or create a folder on your PC.

### Step 4: Start the archiving process

- Click the "Start" button.
- The app will download videos one by one.
- It will then generate summaries, transcripts, and tags automatically.
- You will see progress updates as each video finishes.

### Step 5: Find your notes and videos

- When done, open the output folder you chose earlier.
- You will see markdown files named after the videos.
- Each file contains a summary, transcript, and tags.
- The downloaded videos may also appear here.
